문제3.

HTML5 문서(컨텐츠 포함)를 아래 조건을 적용하여 자유롭게 작성하고, 웹브라우저 실행결과를 출력하시오.
  1. 레이아웃을 위한 HTML5 요소 (실질적인 컨텐츠 내용과 용도에 맞게 작성)
    header, nav(hyperlink), section, article, aside, footer(&특수이름 or 진수코드)
  2. 본문에서 "본인이름_학번"을 클릭할 경우
    웹 페이지가 현재의 웹 페이지 안에서 출력 (창 삽입)
  3. poster 속성을 사용하여 비디오 파일을 11초~17초 재생 구간 지정
    ☞PIP 모드에서 재생된 화면 포함
  4. CSS 적용 방법 중 style 요소(임베디드 방법)을 사용하여
    2가지 이상의 속성을 본문에 적용 (속성은 자유롭게 작성)

창 영역
